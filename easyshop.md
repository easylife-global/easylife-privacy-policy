# EasyShop Privacy Policy

**Last updated:** March 2026

EasyShop ("we", "our", or "us") provides a mobile application that helps small business owners such as retail shops, medical stores, and local businesses manage inventory, sales, and billing.

This Privacy Policy explains how we collect, use, and protect your information when you use the EasyShop mobile application and related backend services.

---

# 1. Information We Collect

EasyShop collects only the information necessary to operate the application.

## Account Information

When you create an account, we may collect:

* Phone number
* Password (stored securely)
* One-time password (OTP) verification for authentication

This information is used only to create and secure your account.

## Business Data

When using the application, you may store business-related data such as:

* Inventory items
* Product information
* Billing records
* Sales transactions

This information is stored securely to provide the functionality of the application.

---

# 2. How We Use Your Information

We use collected information only to provide the services of the EasyShop application, including:

* User authentication and login
* Secure account management
* Inventory management
* Billing and transaction processing
* Application functionality through backend APIs

All actions performed in the app communicate with our backend services using secure API requests.

---

# 3. Payments and Subscriptions

EasyShop offers subscription-based access to certain features.

Subscription purchases are **not processed inside the mobile application**.
When a user chooses to subscribe, they are redirected to a secure web browser page where the subscription purchase is completed.

EasyShop does **not store payment card details**.

---

# 4. Notifications and Advertising

EasyShop:

* Does **not send push notifications**
* Does **not display advertisements**
* Does **not use advertising SDKs**

---

# 5. Data Sharing

We do **not sell, rent, or share personal data with third parties**.

Your information is used only to operate the EasyShop service.

---

# 6. Data Security

We implement reasonable security measures to protect user data including:

* Secure authentication
* Encrypted API communication
* Secure backend infrastructure

Despite these safeguards, no system can guarantee absolute security.

---

# 7. Data Retention

User data is retained for as long as the account remains active and is required to provide the service.

Users may request account deletion by contacting us.

---

# 8. Children's Privacy

EasyShop is intended for business users and is **not designed for children under the age of 13**.
